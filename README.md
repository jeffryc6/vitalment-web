# VitalMente 🌿

**Transformación Integral: Mente, Cuerpo y Espíritu.**

VitalMente es una plataforma diseñada para inspirar y guiar a personas en su camino hacia una vida plena. Enfocada en la resiliencia, la nutrición consciente y la salud mental, combina historias reales de superación con el conocimiento experto de la Dra. Maricel Muñoz.

## ✨ Características

- **Diseño Premium**: Interfaz moderna, responsiva y altamente estética utilizando Tailwind CSS y Framer Motion.
- **Historias de Superación**: Sección dedicada a inspirar a través de casos reales de éxito.
- **Pilares de Salud**: Enfoque multidisplinario en Metabolismo, Mente y Nutrición.
- **Reservas Inteligentes**: Sistema integrado para agendar consultas personalizadas.
- **Arquitectura Robusta**: Construido con Next.js 14 y preparado para integración con Supabase.

## 🛠️ Tecnologías

- **Frontend**: Next.js, React, Tailwind CSS.
- **Animaciones**: Framer Motion.
- **Iconografía**: Lucide React.
- **Backend (Listo para conectar)**: Supabase.

## 🚀 Cómo desplegar en Vercel

1. Crea un repositorio en tu cuenta de GitHub.
2. Sube este código:
   ```bash
   git remote add origin https://github.com/TU_USUARIO/TU_REPOSITORIO.git
   git branch -M main
   git push -u origin main
   ```
3. Ve a [Vercel](https://vercel.com) e importa tu repositorio.
4. ¡Listo! Vercel detectará automáticamente que es un proyecto de Next.js y te dará una URL pública.
